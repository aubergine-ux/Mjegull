# Mjergull
A Simple Weather App with city search, current weather conditions, unit toggle, & location based backgrounds.

Learning Project built with HTML, CSS, & Javascript.

## Features
- Search Any City
- See Current Temperature, Weather Conditons, & Wind Speed in your Unit of Choice.
- Background Photos based on City, from Unsplash.

## Misc.
Used OpenMeteo API for Weather & Unsplash Demo API for Background Photos.
- [Open-Meteo](https://open-meteo.com/)
- [Unsplash](https://unsplash.com/)
